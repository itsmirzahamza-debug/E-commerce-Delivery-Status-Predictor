# E-commerce-Delivery-Status-Predictor
An interactive machine learning application that predicts whether an e-commerce order will be delivered **On Time** or **Late** based on operational metrics. 
## 🚀 Features
- **Exploratory Data Analysis (EDA):** Insights into delivery delay patterns across different shipping types.
- **Predictive Modeling:** A pipeline-based Random Forest Classifier trained on key features (excluding target leakage variables).
- **Interactive Web App:** A clean Streamlit frontend where users can enter real-time order details to get predictions with model confidence.
